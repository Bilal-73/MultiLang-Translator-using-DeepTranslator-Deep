# MultiLang Translator using GoogleTranslator from deep_translator

## Description

MultiLang Translator is a simple and user-friendly web application built with Flask that enables users to translate text between multiple languages using the GoogleTranslator from the `deep_translator` Python library.

The app supports translation among six popular languages: English, German, French, Spanish, Italian, and Hindi. It displays both the original input text and the translated output on the web page.

---

## Features

- Translate text between 6 supported languages.
- Easy-to-use web interface.
- Shows both input and translated text.
- Built with Flask and `deep_translator`.
- Lightweight and quick to run locally.

---

## Supported Languages

- English
- German
- French
- Spanish
- Italian
- Hindi
